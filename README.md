# Portfolio · Irene Mendoza 🖤⚡

> Personal portfolio built entirely with native web technologies — no frameworks, no dependencies.

[![Portfolio](https://img.shields.io/badge/Live-irenemendoza.dev-CAFF00?style=for-the-badge&logoColor=black)](https://irenemendoza.dev)

---

## ✨ Features

- 🖱️ Custom cursor with `mix-blend-mode: exclusion`
- 🎞️ Scroll-triggered entrance animations via `IntersectionObserver`
- 📱 Fully responsive layout (mobile & desktop)
- 📬 Contact form integrated with Formspree
- 🔤 Typefaces: **Syne ExtraBold** (display) + **Space Mono** (body)
- 🎨 Bold, high-contrast color palette: acid green, coral, electric blue & violet on ink black

## 🚀 Deployment

Deployed on **Vercel** with continuous deployment via GitHub.  
Custom domain managed through **dondominio** → DNS A record pointing to Vercel.

🌐 **[irenemendoza.dev](https://irenemendoza.dev)**

---

## 💻 Run locally

```bash
git clone https://github.com/irenemendoza/my-portfolio.git
cd my-portfolio
```

No build step required. Open `index.html` with Live Server or directly in your browser.
