Aplicación Web Interna - 2025

LEVANTE WAVES HOTEL INTRANET

Se necesitaba una solución para centralizar la gestión operativa de un hotel. Se desarrolló una intranet con autenticación, control de acceso por roles y dashboards personalizados para 8 perfiles de empleado. Incluye módulos de fichaje, permisos, gestión de habitaciones, limpieza y mantenimiento. Disponible con usuarios demo (demo123): director, RRHH, recepcionista, limpieza y mantenimiento.

  • Bootstrap 5
